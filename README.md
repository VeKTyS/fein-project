## Prérequis

Pour pouvoir lancer ce projet il vous faudra réaliser plusieurs étapes dans votre terminal : 

Dans le dossier fein-projet, exécutez la commande
```bash
npm install
```

Ensuite, pour lancer le backend puis exécutez les commandes
```bash
cd backend
node server.js
```

Enfin, pour lancer le frontend ouvrez un autre terminal puis exécutez les commandes
```bash
cd frontend
npm run dev
```
